
# FreshMart Dashboard

A simple and interactive web-based analytics dashboard built using HTML, CSS, JavaScript, and Chart.js.
This project simulates a retail dashboard with login authentication and visual insights.

---

## Features

* Login Authentication

  * Username: `admin`
  * Password: `1234`

* Dashboard Overview

  * Total Sales
  * Profit
  * Days Out of Stock

* Charts and Visualization

  * Bar Chart (Sales)
  * Pie Chart (Category Distribution)

* User Interface

  * Glass-style login box
  * Background image design
  * Clean dashboard layout

* Logout Functionality

---

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js (for data visualization)

## How to Run the Project

1. Download or clone the repository:

   ```
   git clone https://github.com/your-username/freshmart-dashboard.git
   ```

2. Open the project folder.

3. Open `index.html` in any web browser.
