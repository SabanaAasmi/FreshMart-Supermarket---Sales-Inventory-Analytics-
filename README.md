# FreshMart Analytics Dashboard

[Live Demo]:
https://sabanaaasmi.github.io/FreshMart-Supermarket---Sales-Inventory-Analytics-/

## Overview

FreshMart Analytics Dashboard is a lightweight web application that provides a visual summary of retail performance metrics. It features a basic authentication system and an interactive dashboard designed to present key business insights in a structured and user-friendly manner.

This project focuses on demonstrating frontend development concepts, including layout design, styling, and data visualization using charts. It is suitable for beginners who want to understand how dashboards are built using core web technologies.

---

## Key Highlights

* Simple login validation for access control
* Display of important business metrics in card format
* Graphical representation of data using charts
* Clean and minimal user interface
* No backend dependency (fully client-side application)

---

## Functional Modules

### Authentication

* Validates user credentials before granting access
* Displays an error message for invalid login attempts

### Dashboard Metrics

* Total Sales
* Profit
* Days Out of Stock

### Data Visualization

* Bar chart to represent sales data
* Pie chart to show product category distribution

### Session Control

* Logout option to reset the application state

---

## Technology Stack

* HTML for structure
* CSS for styling and layout
* JavaScript for interactivity
* Chart.js library for rendering charts

3. Open the `index.html` file in a web browser.

## Application Flow

1. User opens the application
2. Login page is displayed
3. On successful login, dashboard becomes visible
4. Charts are loaded dynamically
5. User can log out to return to the login screen
